Giới Thiệu Netflix Cookies Checker

Ứng dụng này trích xuất NetflixId từ nhiều định dạng cookie khác nhau và tạo ra link đăng nhập trực tiếp vào tài khoản Netflix.

Web này check cookie và gen link login app Netflix qua link

Tính Năng:

  • Trích xuất NetflixId từ file text, JSON hoặc cookie
  • Kiểm tra tính hợp lệ và thông tin tài khoản
  • Tạo token đăng nhập trực tiếp
  • Xử lý hàng loạt nhiều file cùng lúc
  • Gửi kết quả hợp lệ qua Telegram

Định Dạng Hỗ Trợ:

  • File text chứa cookie
  • File JSON chứa cookie
  • File ZIP chứa nhiều file cookie
  • Chuỗi header có cookie NetflixId
  • File cookie định dạng Netscape

Cài Đặt Telegram:

  1. Tạo bot với @BotFather trên Telegram
  2. Lấy token bot từ BotFather
  3. Mở chat với bot và gửi bất kỳ tin nhắn nào
  4. Lấy Chat ID tại: https://api.telegram.org/bot<YourBOTToken>/getUpdates
  5. Bật tính năng gửi Telegram và nhập thông tin xác thực

Hướng Dẫn Sử Dụng:

  1. Dán cookie hoặc tải từ file ở tab Nhập Đơn
  2. Cấu hình gửi Telegram nếu muốn (tùy chọn)
  3. Chọn chế độ xuất dữ liệu (Thông Tin Đầy Đủ hoặc Chỉ Token)
  4. Nhấn "Tạo Token" để xử lý
  5. Để xử lý nhiều file, dùng tab Xử Lý Hàng Loạt

Lưu ý: Công cụ này chỉ dành cho mục đích học tập và nghiên cứu.
